# ZDAI
ZDAI: Programa desenvolvido para automatização de solicitações e ordens de serviço da ZDAlimentos através de inteligência artificial e integração com diferentes tecnologias.
O fluxo é baseado em um estudo de solicitação de serviço através de Inteligência Artificial que ilustra qual profissional é o mais adequado com base na criticidade, competência e nível de cargo. Dessa forma, o banco de dados da empresa é alterado automaticamente através de automação sem que seja necessário trablho manual por parte do Programador de Manutenção da empresa.
